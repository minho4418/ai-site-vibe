# AI 뉴스 — 디자인 시스템

> "시스템 먼저" 원칙(영상 벤치마킹 + CLAUDE.md 안티-제너릭 가드레일)을 한 장으로 모은 토큰 문서.
> 카드·배지·버튼을 바꿀 땐 **여기 정의된 토큰만** 쓴다. 새 색/그림자를 즉흥적으로 만들지 않는다.

## 1. 컬러

### 브랜드
| 역할 | 값 | 용도 |
|------|----|----|
| 브랜드 그라데이션 | `from-fuchsia-600 via-purple-600 to-indigo-600` | 히어로, 로고 |
| 로고 그라데이션 | `from-fuchsia-500 via-purple-500 to-orange-400` | 헤더 로고 칩 |
| 액센트(Primary) | `violet-600` / dark `violet-400` | 링크 호버, 포커스 링, 북마크 활성 |
| 좋아요(Like) | `rose-500/600` | 좋아요 활성·포커스 |

| 역할 | Light | Dark |

### 카테고리 캔디 컬러 (categories.ts 가 단일 출처)

- 칩(이미지 위): 흰/검정 반투명 배경 + 카테고리색 **글자·링**으로만 구분 (`CATEGORY_COLORS`)
- 필터칩 활성: 카테고리색 비비드 배경 (`CATEGORY_CHIP_ACTIVE`)

## 3. 형태 / 간격 / 그림자
- 라운드: 칩 `rounded-full` · 카드 `rounded-2xl` · 큰 섹션 `rounded-3xl`
- 테두리: 카드·인터랙티브는 `border-2` (얇은 1px 금지 — 두꺼운 보더가 브랜드 톤)
- 그림자(레이어드, 색 틴트): `shadow-md` 단독 금지
  - 카드 기본 `shadow-[0_2px_10px_-4px_rgba(15,23,42,0.1)]`
  - 카드 호버 `shadow-[0_18px_40px_-14px_rgba(124,58,237,0.4)]` (보라 틴트)
- 그리드 갭: `gap-5`, 카드 그리드 `1 → sm:2 → lg:3` 열

## 4. 배지 규칙 ⭐ (이번 벤치마킹의 핵심 — 인기/신선도 신호)
> HN(점수)·Futurepedia(사용수치)에서 확인: **인기·신선도는 스캔의 1차 신호**다. 카드 썸네일 **우상단**에 얹는다.

| 배지 | 조건 | 스타일 | 근거 |
|------|------|--------|------|
| 🔥 **인기(HOT)** | **표출된 피드(최신 N개)의 상위 20% 자리**를 조회수 높은 순으로 채움 (조회 0 글 제외 → 실제 수 = min(피드의 20%, 조회된 글 수)) | 앰버→오렌지 그라데이션 pill, 흰 글자 | 사회적 증거(HN 점수/Futurepedia 사용수치) |
| 🆕 **NEW** | 발행 후 **3시간** 이내 | violet 반투명 pill | 신선도(HN/Neuron "N시간 전") |

- 좌상단은 **카테고리 칩** 고정(기존 유지) → 배지는 **우상단**에 세로 정렬.
- HOT 판정은 `HomeClient.tsx`(`HOT_TOP_FRACTION`)에서 전체 최신 집합 기준으로 계산해 `hot` prop 으로 내려준다. 조회수는 카드 클릭 시 `recordView` 로 누적(+화면엔 낙관적 +1 즉시 반영). NEW(`NEW_WINDOW_MS`)는 `ArticleCard.tsx` 상수.

> **왜 절대 임계값(예: 좋아요 10, 24h)이 아니라 상대/좁은 기준인가 — 실데이터 검증 결과:**
> 매일 cron 수집이라 노출 기사가 전부 5~23시간 전 → 24h NEW 면 **모든 카드**에 붙어 무의미.
> 신규 사이트라 좋아요도 대부분 0 → 절대 임계값 HOT 은 안 켜지거나 한꺼번에 켜짐.
> 그래서 HOT 은 **상대 상위**(가짜 인기 방지·자동 활성), NEW 는 **3h "방금 들어온"**(평소 0, 수집 직후만 점등)으로 설계.

## 5. 클릭 유도 (Affordance)
- 카드 전체를 `<a>`로 감싸지 않는다(내부에 좋아요·북마크 버튼이 있어 중첩 금지).
- 대신 **썸네일·제목을 링크**로 두고, 호버 시:
  - 썸네일 하단에 **"읽기 →"** 오버레이 노출 (`opacity`+`translate`만 트랜지션)
  - 제목 색 → `violet-600`, 카드 `-translate-y-1` + 보라 틴트 그림자

## 6. 버튼 상태 (CLAUDE.md 필수 가드레일)
모든 버튼: `hover` / `focus-visible`(2px 보라 링) / `active:scale-[0.95~0.98]` / `disabled:opacity-50` 전부 구현.

## 7. 애니메이션 규칙
- ❌ `transition-all` 금지 → ✅ `transition-[transform,opacity,...]` 처럼 **속성 명시**
- 움직임은 `transform`·`opacity`만. 색은 `transition-colors`.
